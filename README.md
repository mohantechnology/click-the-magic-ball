# Click the Magic Ball
Mini Game using HTML, CSS, and JavaScript

It's a mind game based on reflex and reaction time.
In this game, a random ball will appear and you have to click it as fast as possible.

<b>Time Limit</b> - time given for the current level<br/>
<b>Time Taken</b> - time taken between the ball appearing and the ball being hit<br/>
<b>Score</b> - score till current level<br/><br/>

A slider sets the difficulty level — easy, medium, and hard — incrementing from left to right.
