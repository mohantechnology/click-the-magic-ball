# click-the-magic-ball<br/>
Mini Game using Html , CSS  , Javascript<br/>
<br/>

<a href="https://magic-ball.myselfproject.org" target="_blank"><button  type="button">play the game </button></a>

Its a mind game which is based on  reflex and reaction-time. 
In this game, a random ball will appear and   we have to click the  ball as fast as possible. 

<b> Time Limit</b> - time given for the current level<br/>
<b>Time Taken</b> - time taken between the ball appear and the ball hit<br/> 
<b>Score</b> - score till current level<br/><br/>
A slider which will set the difficulty level  easy, medium and high incrementing from left to right
![magic-ball-hint_compressed](https://github.com/user-attachments/assets/7dd3319f-ccf4-4c91-97ec-c063bc3cbc1a)


Magic ball will come randomly and we have to click it as fast as possible to score high and move to next level. 
Ball will be upgraded in every 5 levels.
![magic-ball-out_compressed](https://github.com/user-attachments/assets/483fbe55-6a14-407c-9e89-e863ceac018a)

 
If time taken to click the ball is less than given time. we lose
